Bantuan

Pertanyaan Umum

Untuk menambahkan pelanggan baru, ikuti langkah-langkah berikut:
  1. Buka halaman Pelanggan
  2. Klik tombol "Tambah Pelanggan"
  3. Isi formulir dengan data pelanggan
  4. Klik "Simpan" untuk menyimpan data pelanggan baru

Untuk melihat laporan pendapatan:
  1. Buka halaman Pendapatan
  2. Pilih rentang waktu yang diinginkan (Minggu Ini, Bulan Ini, atau Tahun Ini)
  3. Lihat ringkasan pendapatan di bagian atas halaman
  4. Untuk laporan lebih detail, klik tombol "Unduh Laporan"

Untuk mengelola gaji karyawan:
  1. Buka halaman Gaji Karyawan
  2. Untuk menambah gaji baru, klik tombol "Tambah Gaji"
  3. Untuk mengedit gaji, klik tombol "Edit" pada baris karyawan yang ingin diubah
  4. Untuk menghapus data gaji, klik tombol "Hapus" pada baris yang ingin dihapus
